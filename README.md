NutriTrack  Fitness & Nutrition Dashboard:-
NutriTrack is a modern, responsive fitness and nutrition tracking application built with React 19 and Vite. Designed to help users monitor their health goals and maintain a consistent, healthy lifestyle, this project focuses on a clean and interactive user experience. 

🚀 Key Features
Fitness Dashboard:A centralized dashboard that provides a clear overview of daily health statistics. 
Fasting Timer: An integrated, easy-to-use timer feature designed specifically to track intermittent fasting schedules.
Progress Tracking: Dynamic rings and widgets that allow users to visualize their health and fitness progress.
Responsive UI: A modern design that includes built-in support for dark mode, ensuring usability across different devices. 

🛠️ Tech StackFrontend: React 19, Vite.
Styling: Custom CSS (with Dark Mode compatibility). 
Development Tools: VS Code, Git, and GitHub. 

📦 Getting Started To run this project locally on your machine
cd nutritrack

**Install dependencies**:
    npm install
    
*Run the application*
npm run dev
